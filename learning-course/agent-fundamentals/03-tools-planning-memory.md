# Tools、Planning、Memory 各自解决什么问题

难度：入门到中级
预计阅读时间：8-10 分钟
建议方式：必须读

很多人会把这三个词混在一起，其实它们解决的是不同层面的问题。

## 1. Tools

工具解决的是：

“Agent 有什么外部行动能力？”

例如：

- 读文件
- 写文件
- 跑命令
- 搜索网页

没有工具，Agent 只能“说”；  
有了工具，Agent 才能“做”。

## 2. Planning

计划解决的是：

“复杂任务怎么拆成多个步骤？”

没有 planning，Agent 很容易：

- 一步想太多
- 跳步骤
- 漏条件

## 3. Memory

记忆解决的是：

“系统如何在多轮中保持连续性？”

没有 memory，Agent 很容易：

- 忘记之前做过什么
- 重复劳动
- 丢失用户偏好和项目上下文

## 一句话关系

- Tools 决定“能做什么”
- Planning 决定“先做什么后做什么”
- Memory 决定“不要忘记什么”
