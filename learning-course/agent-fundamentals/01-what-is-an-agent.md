# 什么是 Agent，和 Chatbot 有什么不同

## 最简单的定义

Chatbot 更像：

- 你问一句
- 它答一句

Agent 更像：

- 你给它一个目标
- 它会自己决定接下来做什么
- 可能会调用工具
- 可能会分步骤推进
- 可能会在外部世界产生动作

## Agent 多出来的东西是什么

至少有 4 个：

1. 目标导向
2. 工具调用
3. 状态延续
4. 自主推进

## 为什么企业里大家会把这两者混淆

因为表面上看，它们都在“和模型对话”。

但工程上完全不是一回事：

- Chatbot 重点是回答质量
- Agent 重点是过程控制

## 一句话区分

Chatbot 主要解决“回答你”  
Agent 主要解决“替你推进任务”
