# 图解总览

这里收录的是整套课程的配套图解文档。它们不是装饰，而是帮助你从“读代码”切换到“看结构”。

## 文档列表

- [启动链路图](./startup-flow.md)
- [运行时循环图](./runtime-loop.md)
- [扩展机制图](./extension-architecture.md)

## 使用建议

### 在学习前看

先看图，建立骨架，再读代码。这样不容易在细节里迷路。

### 在学习中看

每读完一个关键文件，就回来看图，确认自己理解的节点和边是否成立。

### 在学习后看

尝试闭上文档，自己重新画一遍。如果你能用自己的语言重建结构，说明课程已经变成了你的理解。
