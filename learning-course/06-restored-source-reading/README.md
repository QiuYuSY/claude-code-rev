## 小白先看

如果你是第一次读恢复版仓库，先只记住 3 句话：

1. 这不是原始 pristine 仓库，而是恢复版
2. 先抓主干结构，不要先钻 shim 细节
3. 你不需要把所有兼容逻辑都看懂，先看大图
