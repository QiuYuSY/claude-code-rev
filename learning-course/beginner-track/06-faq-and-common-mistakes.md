# 常见问题与误区

难度：入门
预计阅读时间：5 分钟
建议方式：可选

## 问：我是不是必须先懂所有 Agent 概念？

答：
不用。你只要先懂：

- Message
- Tool
- Query Loop
- AppState

就可以开始了。

## 问：为什么我一看源码就觉得很复杂？

答：
因为你看到的是“成熟系统的内部结构”，不是教程项目。复杂是正常的，不是你不行。

## 问：我是不是应该先看 `main.tsx`？

答：
不建议。先看课程、图解和较小的入口文档。

## 问：我是不是要把每个文件都读完？

答：
不用。先抓主线，再补细节。

## 问：哪些内容我可以先跳过？

可以先跳过：

- 太深的 feature gates
- 某些 ant-only 分支
- `shims/`
- `vendor/`
- 特别长的异常分支

## 问：什么时候算“入门成功”？

如果你能说清楚下面 4 句话，就已经成功了：

1. Claude Code 是一个 coding agent 运行时
2. 它不是单次模型调用，而是 query loop
3. REPL 是核心，不是外壳
4. 工具、消息、权限和状态是连在一起的
