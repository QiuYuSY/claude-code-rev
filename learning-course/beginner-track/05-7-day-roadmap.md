# 7 天入门路线

这是一条尽量不吓人的学习路线。

## 第 1 天

读：

- `beginner-track/01-what-is-claude-code.md`
- `beginner-track/02-one-request-explained-simply.md`

目标：

- 先知道它是什么
- 先知道一次请求怎么走

## 第 2 天

读：

- `beginner-track/03-four-core-concepts.md`
- `00-orientation/02-core-concepts-glossary.md`

目标：

- 先把关键概念对上号

## 第 3 天

读：

- 第一课概览
- 第一课流程分析

目标：

- 只搞懂“它怎么启动”

## 第 4 天

读：

- 第三课概览
- 第三课流程分析

目标：

- 只搞懂“REPL 和状态为什么这么重要”

## 第 5 天

读：

- 第二课概览
- 第二课流程分析

目标：

- 只搞懂“query loop 是干嘛的”

## 第 6 天

读：

- 第四课概览
- 第五课概览

目标：

- 先知道权限、消息、扩展为什么重要

## 第 7 天

做：

- 回看三张图
- 写一页自己的总结

总结模板：

- Claude Code 是什么
- 最核心的 3 个设计
- 我现在还不懂的 3 个点
