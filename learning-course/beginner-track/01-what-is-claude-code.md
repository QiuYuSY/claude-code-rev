# Claude Code 是什么

先不要看源码，先把这个问题讲清楚。

## 最简单的理解

Claude Code 不是普通聊天机器人。

它更像一个：

“在终端里运行、能读写文件、能跑命令、能调用工具、能维持长期会话状态的 coding agent。”

## 它和 ChatGPT 式聊天有什么不同

普通聊天更像：

- 你问一句
- 它答一句

Claude Code 更像：

- 你下达一个任务
- 它会思考
- 可能会调用工具
- 可能会读文件、改文件、跑命令
- 会在一个会话里继续推进任务

## 它和 IDE 插件有什么不同

IDE 插件通常更像：

- 在编辑器里帮你补全、解释、改代码

Claude Code 更偏：

- 整个终端工作流里的 agent
- 不只操作代码，还操作会话、任务、权限和工具

## 它和普通 agent 有什么不同

你可以把 Claude Code 看成：

- 一个带工具的 agent
- 一个带状态的会话系统
- 一个带扩展机制的平台

它不只关心“模型答得好不好”，还关心：

- 运行过程怎么控制
- 权限怎么约束
- 状态怎么保存
- 扩展怎么治理

## 小白只需要先记住 3 句话

1. Claude Code 是 coding agent，不是普通聊天机器人
2. 它的核心是“会调用工具并持续推进任务”
3. 它背后有一整套运行时，不只是一个 prompt
