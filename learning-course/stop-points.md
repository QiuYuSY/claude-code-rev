# 学到哪里可以先停

不是每个人都要把整套课全部读完。

这份文档就是告诉你：

“学到哪里，已经足够了。”

## 停点 1：读完 `agent-fundamentals`

你已经能得到什么：

- 理解 Agent 和 Chatbot 的区别
- 理解 Agent loop
- 理解 Tools / Planning / Memory / HITL / Multi-Agent 的基本概念

适合谁：

- 想学 Agent 设计原理的人
- 暂时不想深入 Claude Code 源码的人

## 停点 2：读完 `beginner-track`

你已经能得到什么：

- 不会再把这个仓库当成“一个大杂烩”
- 知道应该先看什么、先跳过什么
- 能理解一次请求在系统里的大致路径

适合谁：

- 小白
- 想先建立感觉、暂时不做深读的人

## 停点 3：读完 `beginner-track + 第 1/2/3 课`

你已经能得到什么：

- 理解 Claude Code 主骨架
- 理解它怎么启动
- 理解 query loop 的角色
- 理解 REPL 和状态系统为什么重要

适合谁：

- 想知道这项目“主干长什么样”的人

## 停点 4：读完 `01-06` 全部正式课程

你已经能得到什么：

- 能做 Claude Code 的结构化源码分析
- 能解释它的核心设计取舍
- 能把关键概念映射到真实模块

适合谁：

- 工程深读
- 技术分享
- 架构分析

## 停点 5：读完 `07-enterprise-agent-report`

你已经能得到什么：

- 能把 Claude Code 的设计抽成企业汇报语言
- 能给管理层、架构评审和方案讨论输出材料

适合谁：

- 方案汇报
- 企业内部平台设计

## 一句话建议

如果你主要想学 Agent，停在 `agent-fundamentals + beginner-track` 就已经很值；  
如果你还想看“这些概念如何落地”，再继续学正式课程。
