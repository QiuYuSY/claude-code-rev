# 这个案例在课程各模块里分别发生了什么

## 在 `agent-fundamentals`

你会学到：

- 这属于一个典型 Agent loop
- 工具为什么会被调用
- 为什么需要状态和反馈

## 在 `beginner-track`

你会学到：

- 一次请求的大致路径
- 为什么 REPL 不是聊天窗口

## 在第一课

你会学到：

- 这个请求是在什么启动结构下进入 REPL 的

## 在第二课

你会学到：

- query loop 如何决定是否去读文件、调工具
- 工具结果为什么要回到消息里

## 在第三课

你会学到：

- REPL 如何把这个过程完整显示出来
- AppState 为什么会记录这些状态

## 在第四课

你会学到：

- 这次任务里的消息是怎么建模的
- 上下文是怎么拼给模型的

## 在第五课

你会学到：

- 如果这个任务不是只靠内建工具，而是要靠 MCP / Skills / Plugins，会怎么扩展

## 在第六课

你会学到：

- 如何区分这个案例里哪些是架构主干，哪些只是恢复版兼容噪音
